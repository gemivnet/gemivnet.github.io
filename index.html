Personal Projects

W1GIV Auctions

W1GIV Auctions is a web based auction management system created to aid in in-person auctions. It allows for the registration of attendees, the entering of items sold during the auction, and checkout which automatically calculates buying/selling totals and commissions. The program also allows for emailing of receipts, contacting attendees which signed up for email notifications, and generating reports.

Speed Sorter

Speed Sorter is an Android and iOS game which is currently in development. The game features an infinite belt of items which need to be sorted into combos featuring power ups and abilities.
